# hyprwatch

![](./.assets/show.png)

## Usage

### Monitor Mode

```sh
mkdir -p $HOME/.cache/hyprwatch/daily_data
hyprwatch -d &> /dev/null &
```

### Show Stats

```sh
hyprwatch --show
```
