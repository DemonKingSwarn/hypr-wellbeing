# hypr-wellbeing

![](./.assets/show.png)

## Usage

> [!NOTE]
> make sure you ran this command before running the program

```sh
mkdir -p $HOME/.cache/hyprwatch/daily_data
```

### Monitor Mode

```sh
hyprwatch -d &> /dev/null &
```

### Show Stats

```sh
hyprwatch --show
```
